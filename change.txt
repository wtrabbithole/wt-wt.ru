Серверное обновление (01.10.19)

    Leopard 2K  — исправлен тип модификации ночного видения с тепловизионной на инфракрасную. 
    Centauro, Centauro ROMOR, Begleitpanzer 57  —  исправлен тип модификации ночного видения с инфракрасной на тепловизионную.
    AMX-30  — добавлена модификация ИК системы ночного видения. 
    Shot Kal Dalet, Type 61, Type 87, Type 75, Type 75 MLRS  — добавлена модификация ИК-системы ночного видения для мехвода. 
    M8 LAC  — добавлена стабилизация с помощью плечевого упора.
    M60A1 (AOS)  — уточнена (увеличена) масса танка с 47,6 до 48 тонн. Источник: TM 9-2350-215-34-2-1.
    Centurion Mk.10  — увеличена толщина дополнительной плиты на ВЛД с 44 до 51 мм. Снижена толщина бронирования маски с 200 до 152 мм. Источник: Centurion 7,8,9,10 (fully modified) armour details, F.V.231921. Centurion 7,8 plate thickness, F.V.231921 // The Centurion Tank, Pat Ware, Brian Delf // Centurion Tank - Bill Munro, 2005
    T-80U  — изменён показатель минимальной кратности прицела. Кратность раньше: 3,6-12х. Новая кратность: 2,7-12х. Источник: Сайт производителя // Танк Т-90С, инструкция по эксплуатации (аналогичный прицел 1Г46)
    M247  — увеличена скорость горизонтального наведения башни с 60 до 90 градусов в секунду. Исправлена кратность прицела. Кратность ранее: 1,5-6х. Новая кратность - фиксированная 12х. Источник: Manprint Analysis of the DIVAD System: Volume II. Lessons Relearned
    3,7 cm Pzgr. (Flak 36, Flak 43)  — увеличена начальная скорость с 740 до 770 м/с. Источник: Einbaumappe 3,7 cm BK-43, Marz 1944 // Documentation W127.
    T25, M36  — исправлена ошибка в обозначении дефолтного бронебойного снаряда. Дефолтным снарядом для указанной техники является M77.
    105 мм снаряд T32  — уточнена (снижена) скорость с 975 до 914 м/с. Источник: RAC - Technical Situation Report No. 35, 1945 // Hunnicutt: Firepower - A History of the American Heavy Tank Data Sheet for Guns 105mm T5E1 and T5E2.
    Type 63-I  — исправлена ошибка в окне модификаций, когда модификация кумулятивного снаряда выходила за его пределы.
    Звуковые события взрывов и пушечных выстрелов получили более яркую атаку.
    Оптимизировано использование виртуальных звуковых каналов аудио-движка.
    Немного откорректирован баланс громкостей орудий наземной техники, а также взрывов и двигателей некоторых самолетов.
    Исправлена проблема перепутанных местами мужского и женского голосовых пакетов в меню настроек звука.
    У пользователей консолей все рецепты по обмену предметов из трофеев с камуфляжами собраны в одну вкладку Мастерской. Выбор рецептов, относящихся только к одному из трофеев, осуществляется через выпадающий список. 
    На страницу Биржи добавлены фильтры для поиска камуфляжей из трофеев "Ночное зрение" и  "Императорский флот". Список фильтров будет актуализироваться по мере добавления новых трофеев.

Патчноут отражает ключевые изменения в игре и не содержит список всех исправлений. Изменения, направленные на развитие War Thunder, производятся регулярно и могут не требовать выхода обновления.
